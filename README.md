## 中身  
[おせろの定石](http://blog.livedoor.jp/sensuikan1973-hamlite/)で示す方針に従ったデータです(改良や追加多数)  
現時点(2017/06/04)では以下が登録済み   
* 「黒番虎大量」 0, +2(一部未登録), +4(大部分未登録)  
* 「黒番闘牛」 0, +2, +4(大部分未登録)  
* 「白番」 +2, +4(大部分未登録), +6(大部分未登録)    
注意：白番の対引分分岐まわりは意図的に除外しています

## アプリに入れる方法  
注意：**自作のデータはバックアップを取っておいて下さい**  
1) opening.db をクリック  
2) Downloadボタンで任意の場所に保存  
3) 端末のdoneX/opening.dbをダウンロードしたものに置換
