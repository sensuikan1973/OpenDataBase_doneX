# doneX用の公開データベースファイル  

## 中身  
[おせろの定石](http://blog.livedoor.jp/sensuikan1973-hamlite/)で示す方針に従ったデータです。(一部改良,追加)  
現時点では以下のような棋譜が登録済み。   
* 黒番虎大量0,+2  
* 黒番闘牛0,+2  
* 白番コンポス受け  
注意：白番の対引分は意図的に除外しています  

## 今後追加予定  
随時更新していきます。 
* 縦白番の各種+4  
* 黒番の各種+4  

## アプリに入れる方法  
注意：**自作のデータはバックアップを取っておいて下さい**  
1) opening.db をクリック  
2) Downloadボタンで任意の場所に保存  
3) 携帯端末のdoneX/opening.dbをダウンロードしたものに置換  

## おまけ　データを確認/操作する方法  
1) [DB Browser for SQLite](http://sqlitebrowser.org/)をダウンロード  
2) opening.dbを開く  
3) 視覚的に確認&操作ができる
